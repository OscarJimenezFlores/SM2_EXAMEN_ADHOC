# 📱 Aplicativo Móvil de Gestión de Tickets de Soporte - MDP

Esta solución tecnológica es una aplicación móvil multiplataforma diseñada para centralizar y optimizar la gestión de incidencias técnicas en la **Municipalidad Distrital de Pocollay**. Utiliza un stack moderno basado en **Flutter** para el frontend y **Firebase** para los servicios de backend, permitiendo un registro y seguimiento eficiente de las solicitudes de soporte interno.

---

## Objetivo General
Diseñar e implementar un sistema móvil que permita gestionar de forma centralizada y eficiente los tickets de soporte técnico reportados por el personal de la MDP, asegurando mayor trazabilidad, rapidez en la atención y calidad del servicio.

## Funcionalidades Principales
- **Autenticación:** Acceso seguro para usuarios y administradores.
- **Gestión de Tickets:** Registro de incidencias, seguimiento de estado en tiempo real y notificaciones automáticas.
- **Panel Administrativo:** Herramientas para la gestión, priorización y cierre de incidencias por parte del equipo de TI.
- **Análisis y Reportes:** Generación de estadísticas de atención y exportación de tickets a formato PDF.

## Tecnologías Utilizadas
| Componente | Herramienta / Tecnología |
| :--- | :--- |
| **Framework** | Flutter (Dart) |
| **Backend & Auth** | Firebase Authentication |
| **Base de Datos** | Firebase Firestore (NoSQL) |
| **Notificaciones** | Firebase Cloud Messaging |
| **Arquitectura** | Cliente-Servidor Modular |

## Público Objetivo
- **Usuarios Generales:** Empleados de la MDP que reportan incidencias técnicas.
- **Administradores:** Personal del área de TI encargado de resolver tickets y gestionar el sistema.

## Seguridad y Cumplimiento
- **Control de Acceso:** Basado en roles (RBAC).
- **Privacidad:** Cumplimiento con la **Ley de Protección de Datos Personales (N.º 29733)** de Perú.
- **Calidad:** Diseño bajo estándares **ISO 9126**.

## Impacto Esperado
- **Eficiencia:** Mejora del 30% en la gestión de procesos internos.
- **Rapidez:** Reducción del 25% en el tiempo de respuesta a incidentes.
- **Trazabilidad:** Historial completo desde el registro hasta la solución definitiva.
